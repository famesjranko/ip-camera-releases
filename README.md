# Android IP Camera

Turn your Android device into a network camera with real-time streaming, two-way audio, and a web-based dashboard.

## Features

- **Live Video Streaming** - MJPEG and H.264 (hardware-accelerated)
- **Web Dashboard** - Control camera settings from any browser
- **Two-Way Audio** - Mic streaming + speaker/TTS playback
- **Multi-Camera Support** - Front, back, wide, telephoto, macro
- **Authentication** - Optional password protection
- **Background Streaming** - Keeps streaming when minimized
- **Power Optimization** - CPU throttling, camera standby mode

Download the latest APK from the [Releases](https://github.com/famesjranko/ip-camera-releases/releases) page.

## Requirements

- Android 7.0+ (API 24)
- Camera and microphone permissions

## Installation

1. Download the APK from [Releases](https://github.com/famesjranko/ip-camera-releases/releases)
2. Enable "Install from unknown sources" if prompted
3. Install and grant permissions when requested

## Quick Start

1. Open the app and tap **Start Streaming**
2. Navigate to the URL shown (e.g., `https://192.168.1.100:8443`)
3. Control via web dashboard or Android app
